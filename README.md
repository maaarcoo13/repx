# repx
#Repx is a mobile-first fitness tracking application built using Ionic + Angular, designed to help users create personalized workout plans, log exercises, and store their progress in Firebase.
#This app uses Firestore to provide real-time data synchronization across devices.
#
#Features
#Create workout plans
#Add exercises with reps, sets, weight, and more
#Store all data in Firebase Firestore
#Mobile-friendly UI with Ionic Components
#Real-time updates using AngularFire
#Charting for progress tracking (Chart.js + ng2-charts)
#
#Tech Stack
#Ionic 8
#Angular 20
#Firebase 12 + AngularFire
#Capacitor (Android & iOS support)
#SCSS for styling
#Chart.js + ng2-charts
#
#Prerequisites:
#Node.js v18 or later
#Ionic CLI
#A Firebase Project
#
#Intstallation
#Clone the repository
#Install dependencies
#The enrvironment requires Firebase (this is already included in the code)
#

#@ionic/angular 8
